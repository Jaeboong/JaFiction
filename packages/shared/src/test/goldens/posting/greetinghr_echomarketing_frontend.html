💻프론트엔드 개발자는,

단순한 ​화면 ​구현을 ​넘어, 마케팅 ​솔루션의 사용자 경험(UX)을 설계하는 ​전략가입니다. ​마케터가 마주하는 ​복잡한 데이터와 프로세스를 ​직관적인 UI로 ​풀어내어 ​업무 효율을 ​극대화하고, ​이를 ​통해 조직이 더 ​높은 ​혁신에 집중할 수 ​있는 ​환경을 ​만듭니다.


☑️ 이런 ​도구를 사용하여 ​개발 ​및 소통 ​하고 있습니다.

  • Jira로 ​프로젝트 ​관리
  • Bitbucket으로 형상 관리
  • Confluence, ​Notion, Google ​Drive로 문서 관리
  • AWS Cloud 환경에서 개발
  • Jenkins, code deploy, amplify를 통해 자동 배포
  • MS-Teams, Slack으로 의사소통


☑️ 이런 기술 스택을 사용하고 있습니다.

  • 코어 : React, Typescript, Next
  • 상태관리: Redux Toolkit, React-Query, Zustand, Recoil
  • 스타일링 : emotion, tailwindCSS
  • 패키지 매니저 : Yarn Berry, npm , pnpm(TurboRepo)
  • 클라우드 : AWS(S3, CloudFront, Route53, Amplify), Docker, Kubernetes(Pod 기반 운영)


■ 주요 업무

  • React, Typescript + Next 기반으로 광고운영 및 백오피스 플랫폼을 개발 합니다.
  • monorepo 환경에서 공용 UI 라이브러리 및 FE 개발환경 구축 합니다.
  • 사내 주요 플랫폼의 UI/UX를 유지보수하고 서비스 가치를 고도화합니다.
  • 바이브 코딩 산출물의 중앙화 UI를 설계하고, 마케터의 기획이 실질적인 솔루션이 되도록 UX를 분석 및 개선합니다.
  • AI 에이전트를 활용하여 에이전틱한 개발 환경을 구축하고 다수의 프로젝트를 효율적으로 리딩합니다.


■ 지원 자격

  • 국내외 4년제 대학 졸업(예정)자 이신 분
  • React 또는 Next.js 등을 활용한 모던 FE 프레임워크 사용에 능숙하신 분
  • Javascript와 ES6+에 대한 깊은 이해가 있으신 분
  • Typescript의 이해와 활용 경험이 있으신 분
  • 웹 서비스의 구성 전반에 대한 이해가 있으신 분
  • REST API 와 Network 에 대한 이해가 있으신 분
  • Git 사용에 능숙하신 분
  • 기술을 통해 비즈니스의 문제를 해결하는 데에 관심 있으신 분
  • 신규 기술과 솔루션을 리서치하고 사용하는 것에 적극적이신 분
  • 소속 부서 및 타 부서와의 소통 및 협업이 원활하신 분


■ 이런한 분들을 우대 합니다

  • 컴퓨터 공학 및 유관 학과 전공하신 분
  • Flux 아키텍처에 대한 이해가 있으신 분
  • monorepo 를 구축하고 운영한 경험이 있으신 분
  • 팀 생산성 향상에 대해 고민을 많이 하고, 세부 내용들을 적용해 보신 분
  • UI 라이브러리 및 디자인 시스템을 구축하고 운영한 경험이 있으신 분
  • AI 도구를 활용하여 코드 작성, 리팩토링, 테스트, 문서화 등 개발 생산성을 개선해본 경험이 있으신 분
  • PWA 구축 경험이 있으신 분
  • 기술 공유를 좋아 하시는 분
  • 코드 가독성에 대한 고민을 하시는 분
  • 마케팅 도메인에 대한 배경지식이나 관련 프로젝트 경험이 있으신 분


■ 급여 사항

  • 초봉 4,500만원 (정규직 전환 시 적용 / 수습기간 3개월간은 별도 급여 적용)


■ 근무 조건

  • 고용 형태: 3개월 계약 중 근무 평가를 통해 정규직 전환
  • 입사 예정일 : 5월 초 (*입사 예정일은 상황에 따라 변동될 수 있습니다.)
  • 근무지 : 서울특별시 송파구 올림픽로35다길 42, 한국루터회관 에코마케팅
  • 업무시간 : 주 5일, 10:00 ~ 19:00



함께하게 될 팀은,

  • 고객의 니즈와 문제를 분석하고 기술적으로 해결 합니다. 기술에 제한을 두지 않고 문제 해결에 집중하며 필요한 기술이라면 연구하고 사용하여 기술 범위를 넓히며 성장 할 수 있습니다.
  • 솔루션 개발뿐만 아니라 AWS클라우드의 서비스를 사용하면서 개발자로서 한층 더 성장 할 수 있습니다.
  • 백엔드 개발자, 데이터엔지니어 등 서로 협업하며 함께 성장 할 수 있습니다.



함께 일하게 될 리더의 한마디,

  • React, Typescript + Next 기반으로 임직원들이 매출을 극대화할 수 있는 것이라면 어떤 분야든 백오피스 플랫폼(웹/앱)으로 개발합니다.
  • 사내 모든 문제를 능동적으로 정의하고 해결 방안을 고민하며 해결책이 될 수 있는 플랫폼을 개발하고 제공하는 것을 목표로 하고 있어요.
  • 사내에 서비스를 제공함으로써 임직원들의 매출 극대화에 집중 할 수 있는 환경을 제공합니다.
  • 프론트엔드 개발자로서 React, Typescript, Next를 주로 사용하기 때문에 해당 스택으로 프로젝트 경험이 필수에요.
  • AWS 등 퍼블릭 클라우드를 사용해 본 경험이 있다면 더더욱 좋아요. 또한 기술을 통해 비즈니스의 문제를 해결하는 데에 높은 관심과 신규 기술과 서비스를 리서치하고 사용하는 것에 적극적인 자세가 있다면 좋을 것 같아요.
  • 우리 팀은 앞으로 사내 마케팅 조직의 솔루션 컨설팅 조직이 되려고 합니다. 최고의 프론트엔드 엔지니어가 되고싶다면, 지금 바로 지원해주세요!

  • 전형별 합격자에 한하여 다음 전형일정을 개별 통보합니다.

  • 입사 지원서 최종 제출 후, 수정이 불가하오니 신중하게 검토하신 후 제출해 주시기 바랍니다.
